This is a game that is played entirely in the console. Yes, it's a text-based adventure game.
Do not have the Game Window in unity go full screen; it will be very difficult to play.
If controls are unresponsive, make sure you have the game window selected.
Make sure that the "Collapse" button in the console window is unchecked.
Enabling the "Clear On Play" button on in the console window will help replay the game much easier.
Please don't press buttons simultaneously; you may break the code.

Controls:
Z or X: Choose an Option
C: Use Inventory
	Use the Number keys above the keyboard to use an item.*

*Some items are only useful in certain situations.
